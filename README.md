# java-cli-maven-surefire-jacoco-junit-test-hello-world

## Description
A POC for maven app using JUnit
framework with jacoco and surefire plugins.

## Tech stack
- java
- maven
  - junit
  - jacoco
  - surefire

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- jacoco report under bin/target/site/jacoco

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
